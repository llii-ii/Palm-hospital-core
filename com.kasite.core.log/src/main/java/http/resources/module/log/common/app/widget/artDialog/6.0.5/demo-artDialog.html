对话框使用示例

普通对话框,只有“确定”


            dialog({title:'我是标题党', content:'我是内容', ok: function () {alert("这里是回调事件")} }).showModal();

	   //点击遮罩层自动关闭 quickClose: true

普通对话框，有“确定”和“取消”


            dialog({title:'请注意!', content:'内容内容内容内容内容内容', ok: function (){alert("你点了确定")}, cancel: function () {alert("你点了取消")} }).showModal();

普通对话框，三个自定义按钮

            dialog({
                title:'我是标题君!',
                content:'内容内容内容内容内容内容',
                button: [
                            {value: '普通',callback: function(){} },
                            {value: '二逼',callback: function(){} },
                            {value: '文艺',callback: function(){} }
                        ]
                }).showModal();

普通对话框，多个自定义按钮

            dialog({
                content: 'Hello World!',
                button: [
                            {value: '同意',callback: function () { this.content('你同意了'); return false;} },
                            {value: '不同意',callback: function () {this.content('你不同意');return false;} },
                            {value: '无效按钮', disabled: true},
                            {value: '关闭我'}
                        ]
                }).showModal();
            });

无按钮无标题对话框，定时关闭

           dialog({contentType:'tipsbox', content:'对话框将在三秒内关闭'}).showModal();
           
           //自动关闭时间 closeTime: 3000   （默认三秒）

无按钮无标题对话框，定时关闭(黑色)

           dialog({contentType:'tipsbox', skin:'bk-popup' , content:'对话框将在三秒内关闭'}).showModal();
           
           //自动关闭时间 closeTime: 3000   （默认三秒）

加载中请稍候


 	dialog({contentType:'load'}).showModal();
	
    	//contentType 值为 'load' 默认提示语是 '加载中，请稍候...'  , 可自行修改 content 的 值； 例如 ： content : '列表数据载入中，请稍候...'

数据载入中请稍候

关闭对话框的方法：close(), 移除对话框方法：remove()


            var d = dialog({contentType:'jsonload'}).showModal();
            
            //数据载入完 关闭:   d.close().remove();
            
            //contentType 值为 'jsonload' 默认提示语是 '数据载入中，请稍候...'  , 可自行修改 content 的 值； 例如 ： content : '列表数据载入中，请稍候...'

更详细的使用示例请参见官方文档
